# CRUD

A application demonstrating **C**reate, **R**ead, **U**pdate, and **D**elete operations using Angular

## Technologies

- **[Angular](https://angular.io)**
- **[Angular Material](https://material.angular.io)**
- **[Angular Apollo Graph](https://the-guild.dev/graphql/apollo-angular)**
- **[NGRX](https://ngrx.io)**

## How to deploy

Run the following commands in your terminal

```bash
npm install
ng serve
```

> [!NOTE]
> This project uses GraphQL, which simplifies data handling by eliminating the need for traditional models and parsers. Data structure is defined using **schemas** as outlined in the [schemas documentation](https://graphql.org/learn/schema/), and data fields can be renamed using **aliases**, as explained in the [aliases guide](https://graphql.org/learn/queries/).
